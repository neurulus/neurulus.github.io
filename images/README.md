# MICHRO Images Directory

This directory contains all product and website images for the MICHRO website.

## Directory Structure

```
images/
├── products/          # Product-specific images
│   ├── neural-3000-hero.jpg
│   ├── kinetic-pro-hero.jpg
│   ├── quantum-hero.jpg
│   └── core-hero.jpg
├── homepage/          # Homepage images
│   ├── product-neural.jpg
│   ├── product-kinetic.jpg
│   ├── product-core.jpg
│   └── product-quantum.jpg
└── solutions/         # Solution page images (optional)
    ├── gen-ai-factory.jpg
    ├── hft-trading.jpg
    └── modernization.jpg
```

## Image Specifications

### Product Hero Images
- **Dimensions**: 1200x800px
- **Format**: JPG or WebP
- **Quality**: High (90%+)
- **Purpose**: Main product showcase on individual product pages

### Homepage Product Cards
- **Dimensions**: 600x400px
- **Format**: JPG or WebP
- **Quality**: High (85%+)
- **Purpose**: Product cards on homepage

## Placeholder Images

Until images are generated, the HTML files will show gradient placeholders. Once you have the images:

1. Place them in the appropriate directories above
2. Ensure filenames match exactly as specified
3. Optimize images for web (use tools like TinyPNG or ImageOptim)
4. Test that all images load correctly
